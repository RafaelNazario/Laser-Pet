# Projeto Laser Pet


>Projeto Desenvolvido ao final da formação Microcontroladores e Eletrônica Aplicada da empresa Alura.


Foram utilizados neste projeto:


```
1x NodeMCU-ESP8266
1x Sensor de Presença e Movimento PIR
1x Módulo Laser
2x Micro Servo 9G
```
